## 二、一级目录说明

- `rl/`：强化学习实验主目录，包含手搓实现与调库实验。
- `transformer/`：Transformer 相关项目目录。
- `vision_agent/`：视觉 Agent 方向项目目录。
- `papers/`：论文阅读笔记目录。
- `reports/`：阶段性实验总结与学习报告目录。

## 三、二级目录说明

- `cartpole-dqn/`：CartPole 调库 DQN 实验（代码、依赖、实验报告、子 README）。
- `simple-env-qlearning/`：自定义小环境的 Q-learning 实验（训练代码、笔记、趋势图）。
- `ppo-scratch/`：PPO 手搓项目目录（已创建，待实现）。
